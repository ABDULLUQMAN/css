CSS 
   Cascading style sheet .It is used to style the webpages and websites

CSS syntax: 
  selector{
                property:value;
                }
Definition: CSS (Cascading Style Sheets) is used to style and design the HTML content of a webpage. It controls layout, colors, fonts, and spacing.

Purpose: To make web pages attractive and easy to read. To separate structure (HTML) from design (CSS).

Ways to Add CSS: 

Inline CSS: inside the HTML tag using the style attribute.

Internal CSS: inside style  tag in the HTML file.

External CSS: written in a separate .css file and linked using  Link

CSS(Cascading style sheet) is a language designed to simplify the process of making webpages presentaion

DAY-02

Selector

In CSS, selectors are used to target and style specific elements on a webpage. They act like a bridge between the HTML structure and the design rules you want to apply.

DAY-03

Box Model

The box model helps in controlling layout and spacing on a webpage. When you change padding, border, or margin values, you’re adjusting how much space an element takes and how it interacts with nearby elements.

DAY-04

Color-Theory

Color Theory refers to the study of how colors work together to create a visually appealing design. It helps web designers choose the right color combinations to make a webpage attractive.

DAY-05

Typography

Typography refers to the style, arrangement, and appearance of text on a webpage. It plays a key role in making content readable, attractive, and professional.

DAY-06

Background-Properties

Background Properties are used to control the background appearance of an element, such as its color, image, position, and how it repeats or scrolls.

DAY-07

Display-propeties

Display Properties control how elements are shown (or displayed) on a webpage. They define the layout behavior of elements — how they appear next to each other or stack on top of one another

DAY-08

Animations

CSS animation allows elements to change styles smoothly over time without JavaScript. It uses keyframes to define the stages of an animation and properties to control its behavior.

Typography

Definition: Typography deals with how text looks on a webpage (font, size, spacing, etc.)
Common Properties: 
    
font-family: sets font type

font-size: sets text size

font-weight: makes text bold

text-align: aligns text (left, right, center, justify)

line-height: sets space between lines

letter-spacing: space between letters


Example:

p {
  font-family: Arial, sans-serif;
  font-size: 18px;
  line-height: 1.5;
  text-align: justify;
}
